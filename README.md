# 🏥 Hospital Management System (C++)

This is a console-based Hospital Management System developed using Object-Oriented Programming in C++ during my 2nd semester.

## 🚀 Features
- Patient Registration System
- Doctor Management
- Appointment Scheduling
- Billing System
- Patient Record Display

## 🧠 Concepts Used
- Classes & Objects
- Inheritance
- Encapsulation
- Operator Overloading
- STL (Vectors)

## ⚙️ How to Run
1. Compile:
   g++ main.cpp -o hospital
2. Run:
   ./hospital
